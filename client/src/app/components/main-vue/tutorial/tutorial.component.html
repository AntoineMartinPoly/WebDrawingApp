Overview

Welcome! The application is divided in two main section. The toolbar and the drawing view.

Toolbar

The toolbar contains multiple buttons that can be put into 3 categories:

- Tool - Modifier - Option

Toolspurpose is to create elements, such as traces, shapes, texts and stamps.

Modifierspurpose is to modify element. Those tools are the colorizer, the pipette, the selection and the eraser.

Options purpose is to create, save, load and export drawingsand to access tutorial.

To access one of the feature, simply click on it's button

Don't worry about remembering all functionnalities! A hover will give the name of the button

Drawing view

The drawing view is the core of the application. It is where you're allowed to draw and edit element as much as you want. Note that if you move your cursor outside of the drawing view while creating an element, that element will be deleted.
